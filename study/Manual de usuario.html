1. 🎯 Introducción

Bienvenido al Sistema Educativo PAI-REP (Plataforma Académica Institucional República de Panamá), una plataforma integral diseñada para gestionar todos los aspectos académicos de una institución educativa. Este sistema permite la administración completa de estudiantes, profesores, planes académicos, calificaciones, biblioteca y comunicación académica.

🚀 Características Principales

  • Gestión Académica Completa: Planes académicos, materias, actividades y calificaciones
  • Sistema de Biblioteca: Catálogo digital con sistema de préstamos y solicitudes
  • Foro Académico: Espacio de comunicación moderado para la comunidad educativa
  • Multi-Rol: Diferentes interfaces según el rol del usuario
  • Reportes y Estadísticas: Seguimiento detallado del progreso académico
⚠️ Importante: Este manual está diseñado para la versión actual del sistema. Las funcionalidades pueden variar según las actualizaciones y configuraciones específicas de tu institución.

2. 🚀 Primeros Pasos

Acceso al Sistema

Para acceder al sistema, necesitarás tus credenciales proporcionadas por el administrador de la institución.

Pasos para el Primer Acceso:

  1. Dirígete a la URL proporcionada por tu institución
  2. Ingresa tu correo electrónico y contraseña
  3. Haz clic en "Iniciar Sesión"
  4. Serás redirigido al dashboard según tu rol

Navegación General

El sistema cuenta con una navegación intuitiva que varía según tu rol:

Elemento Descripción Disponible para
Dashboard Página principal con resumen de actividades Todos los roles
Perfil de Usuario Información personal y configuración Todos los roles
Cerrar Sesión Salir del sistema de forma segura Todos los roles

3. 👥 Roles y Permisos

El sistema maneja cuatro roles principales, cada uno con permisos específicos:

Administrador ADMIN

  • Gestión completa de usuarios (crear, editar, eliminar)
  • Administración de planes académicos y materias
  • Supervisión de calificaciones y períodos académicos
  • Gestión completa de la biblioteca y préstamos
  • Moderación del foro académico
  • Acceso a reportes y estadísticas generales

Profesor TEACHER

  • Gestión de actividades académicas
  • Calificación de estudiantes
  • Manejo de períodos académicos
  • Consulta de planes académicos
  • Participación en el foro académico
  • Acceso a la biblioteca

Estudiante STUDENT

  • Visualización de materias y calificaciones
  • Consulta de actividades académicas
  • Solicitud de préstamos bibliotecarios
  • Participación en el foro académico
  • Acceso a su progreso académico

Padre/Acudiente PARENT

  • Consulta del progreso académico de sus hijos
  • Visualización de calificaciones
  • Acceso a información general
  • Participación limitada en el foro

4. 👑 Manual del Administrador

Como administrador, tienes acceso completo a todas las funcionalidades del sistema. Tu dashboard muestra un resumen general de la actividad de la plataforma.

4.1 Gestión de Usuarios

Crear un Nuevo Usuario

  1. Ve a "Gestión de Usuarios" en el menú principal
  2. Haz clic en "Agregar Usuario"
  3. Completa el formulario con la información requerida:
    • Nombres y apellidos
    • Correo electrónico (será el usuario de acceso)
    • Documento de identidad
    • Teléfono y dirección
    • Fecha de nacimiento
    • Rol del usuario
    • Para estudiantes: asignar grado y grupo
  4. Haz clic en "Crear Usuario"
⚠️ Importante: Al crear un estudiante y asignarle un grupo, el sistema automáticamente creará su inscripción activa. No es necesario crear la inscripción manualmente.

Gestión de Roles

Rol Cuándo Asignar Campos Adicionales
admin Personal administrativo Ninguno
teacher Docentes Ninguno
student Estudiantes activos Grado y Grupo (obligatorios)
parent Padres/acudientes Ninguno

4.2 Planes Académicos

Los planes académicos son la estructura central del sistema educativo. Definen qué materias se imparten en cada grado.

Crear un Plan Académico

  1. Ve a "Planes Académicos"
  2. Haz clic en "Crear Plan Académico"
  3. Completa la información:
    • Nombre: Nombre descriptivo del plan
    • Descripción: Detalles del plan académico
    • Año Académico: Año al que corresponde
    • Fecha de Inicio y Fin: Período de vigencia
  4. Guarda el plan académico

Asignar Materias a un Plan

  1. Desde la lista de planes académicos, haz clic en "Ver Detalles"
  2. Ve a la sección "Materias del Plan"
  3. Haz clic en "Agregar Materia"
  4. Selecciona las materias que deseas incluir
  5. Para cada materia, define:
    • Créditos: Valor académico de la materia
    • Horas semanales: Intensidad horaria
    • Es obligatoria: Si es requerida o electiva

4.3 Gestión de Materias

Crear una Nueva Materia

  1. Ve a "Materias" en el menú
  2. Haz clic en "Nueva Materia"
  3. Completa la información:
    • Nombre: Nombre de la materia
    • Código: Código único identificador
    • Área: Área académica (Matemáticas, Ciencias, etc.)
    • Descripción: Descripción detallada
    • Créditos: Valor en créditos

Asignar Profesores a Materias

Las asignaciones se realizan por grupo específico:

  1. En la vista de materias, selecciona una materia
  2. Ve a la sección "Asignaciones"
  3. Haz clic en "Nueva Asignación"
  4. Selecciona:
    • Profesor: Docente que impartirá la materia
    • Grupo: Grupo específico al que se asigna
    • Período: Período académico correspondiente

4.4 Sistema de Biblioteca

El módulo de biblioteca permite gestionar el catálogo de recursos y los préstamos a estudiantes.

Gestión de Recursos

Agregar un Nuevo Recurso
  1. Ve a "Biblioteca" → "Gestionar Recursos"
  2. Haz clic en "Agregar Recurso"
  3. Completa el formulario:
    • Título: Nombre del recurso
    • Autor: Autor del recurso
    • ISBN: Código ISBN (opcional)
    • Tipo de Recurso: Libro, Revista, Digital, Multimedia
    • Descripción: Descripción detallada
    • Ubicación: Ubicación física en la biblioteca
    • Copias Totales: Cantidad total de ejemplares
    • Estado: Activo/Inactivo para préstamos
✅ Tip: Las copias disponibles se calculan automáticamente restando las copias actualmente prestadas del total de copias.
Editar un Recurso
  1. En la lista de recursos, haz clic en "Editar"
  2. Modifica los campos necesarios
  3. Para cambiar el estado (activo/inactivo), usa el checkbox correspondiente
  4. Al modificar las copias totales, el sistema validará que no sean menores a las actualmente prestadas

Gestión de Préstamos

Sistema de Solicitudes

Los estudiantes no pueden tomar libros directamente. Deben solicitar préstamos que requieren tu aprobación:

Proceso de Aprobación de Préstamos
  1. Ve a "Biblioteca" → "Gestionar Préstamos"
  2. Revisa las "Solicitudes Pendientes"
  3. Para cada solicitud puedes:
    • Aprobar: El préstamo se activa automáticamente
    • Rechazar: Puedes agregar un motivo de rechazo
  4. Al aprobar:
    • Se asignan las fechas de préstamo y devolución
    • Se reduce el inventario de copias disponibles
    • Se notifica al estudiante
Monitoreo de Préstamos

El sistema proporciona varias vistas para monitorear los préstamos:

Sección Descripción Acciones Disponibles
Solicitudes Pendientes Préstamos que requieren aprobación Aprobar, Rechazar
Préstamos Activos Préstamos en curso Ver detalles, Marcar como devuelto
Préstamos Vencidos Préstamos que superaron la fecha de devolución Contactar estudiante, Procesar devolución
Devoluciones Recientes Préstamos devueltos recientemente Ver historial

4.5 Moderación del Foro

El foro académico es un espacio moderado donde la comunidad educativa puede intercambiar ideas académicas.

Panel de Moderación

  1. Ve a "Foro" → "Panel de Moderación"
  2. Revisa el contenido pendiente de aprobación
  3. Para publicaciones pendientes:
    • Lee el contenido completo
    • Verifica que cumple las políticas académicas
    • Aprobar: La publicación se hace visible
    • Rechazar: La publicación se oculta
  4. Para comentarios pendientes: mismo proceso

Gestión de Contenido

🛡️ Políticas de Moderación Recomendadas
  • Contenido Académico: Solo aprobar contenido relacionado con temas educativos
  • Respeto: Rechazar contenido ofensivo o irrespetuoso
  • Relevancia: Verificar que el contenido sea relevante para la comunidad
  • Calidad: Promover discusiones constructivas y bien fundamentadas

5. 👨‍🏫 Manual del Profesor

Como profesor, tienes acceso a las herramientas necesarias para gestionar tus clases, crear actividades y calificar a tus estudiantes.

5.1 Gestión de Actividades

Crear una Nueva Actividad

  1. Ve a "Materias" y selecciona la materia donde quieres agregar la actividad
  2. Haz clic en "Nueva Actividad"
  3. Completa el formulario:
    • Nombre: Título descriptivo de la actividad
    • Descripción: Detalles de qué se debe realizar
    • Tipo de Actividad: Examen, Tarea, Proyecto, Quiz, etc.
    • Puntaje Máximo: Calificación máxima (ej: 5.0, 10, 100)
    • Porcentaje: Peso de la actividad en la nota del período
    • Fecha de Entrega: Cuándo vence la actividad
    • Período: A qué período académico pertenece
    • Grupos: Selecciona los grupos que participarán
  4. Guarda la actividad
⚠️ Importante sobre Porcentajes: La suma de porcentajes de todas las actividades de un período debe ser exactamente 100% para poder finalizar el período. El sistema te avisará si falta completar el 100%.

Estados de las Actividades

Estado Descripción Acciones Disponibles
Borrador Actividad en preparación, no visible para estudiantes Editar, Publicar, Eliminar
Publicada Visible para estudiantes, pueden trabajar en ella Editar, Calificar, Finalizar
Finalizada Ya no acepta cambios, calificaciones cerradas Ver resultados, Reabrir

5.2 Sistema de Calificaciones

Calificar una Actividad

  1. Ve a la actividad que quieres calificar
  2. Haz clic en "Calificar Estudiantes"
  3. Tienes dos opciones:
    Calificación Individual:
    • Haz clic en "Calificar" junto al nombre del estudiante
    • Ingresa la calificación (respetando el puntaje máximo)
    • Guarda la calificación
    Calificación Masiva:
    • Usa el formulario de calificación masiva
    • Ingresa las calificaciones de todos los estudiantes
    • Guarda todas las calificaciones de una vez

Sistema de Calificaciones Colombiano

El sistema maneja la escala de calificaciones de 1.0 a 5.0:

Rango Desempeño Color en el Sistema
4.6 - 5.0 Desempeño Superior Verde
4.0 - 4.5 Desempeño Alto Verde
3.0 - 3.9 Desempeño Básico Amarillo
1.0 - 2.9 Desempeño Bajo Rojo

5.3 Manejo de Períodos

Finalizar un Período

Para finalizar un período académico, se deben cumplir dos condiciones obligatorias:

Condiciones para Finalizar:
  1. 100% en Actividades: Todas las materias deben tener actividades que sumen exactamente 100% en porcentajes
  2. Todos los Estudiantes Calificados: Cada actividad debe tener calificaciones para todos los estudiantes inscritos
Proceso de Finalización:
  1. Ve a la sección de "Períodos"
  2. Selecciona el período que quieres finalizar
  3. Haz clic en "Finalizar Período"
  4. El sistema verificará las condiciones:
    • Si hay materias sin 100%: te mostrará cuáles faltan
    • Si hay estudiantes sin calificar: te indicará cuáles actividades
    • Si todo está correcto: procederá con la finalización
✅ Al Finalizar un Período: El sistema calculará automáticamente las notas finales de cada estudiante en cada materia, considerando los porcentajes de las actividades realizadas.

6. 🎓 Manual del Estudiante

Como estudiante, puedes consultar tu progreso académico, acceder a recursos de biblioteca y participar en el foro académico.

6.1 Visualización de Materias

Acceder a Mis Materias

  1. Haz clic en "📖 Mis Materias" en el menú principal
  2. Verás un dashboard con resumen de tu progreso:
    • Total de materias inscritas
    • Materias con calificaciones
    • Tu promedio general actual
    • Total de períodos académicos

Explorar una Materia

  1. Haz clic en cualquier materia para expandir sus detalles
  2. Verás información organizada por períodos académicos:
  3. Para cada período encontrarás:
    • Estado del período: Activo, Finalizado, etc.
    • Nota final del período: Si ya está calificado
    • Lista de actividades: Todas las actividades del período

Ver Actividades y Calificaciones

Para cada actividad puedes ver:

Información Descripción
Nombre y Descripción Título y detalles de qué se debe realizar
Tipo de Actividad Examen, Tarea, Proyecto, etc.
Estado Publicada, Finalizada, etc.
Porcentaje Peso de la actividad en la nota final
Tu Calificación Nota obtenida (si ya está calificada)
Puntuación Puntos obtenidos del total posible

Interpretación de Colores

  • 🟢 Verde (4.0 - 5.0): Excelente desempeño
  • 🟡 Amarillo (3.0 - 3.9): Desempeño básico, puedes mejorar
  • 🔴 Rojo (Menos de 3.0): Necesitas refuerzo académico

6.2 Sistema de Biblioteca

Explorar el Catálogo

  1. Haz clic en "📚 Biblioteca" en el menú
  2. Explora el catálogo de recursos disponibles
  3. Usa los filtros para encontrar lo que necesitas:
    • Buscar: Por título, autor o tema
    • Tipo de Recurso: Libros, Revistas, Digital, Multimedia
    • Disponibilidad: Solo recursos disponibles para préstamo

Solicitar un Préstamo

  1. Encuentra el recurso que quieres solicitar
  2. Verifica que esté disponible (copias disponibles > 0)
  3. Haz clic en "Solicitar Préstamo"
  4. Recibirás un mensaje de confirmación
  5. Importante: Tu solicitud debe ser aprobada por el administrador de la biblioteca
⚠️ Proceso de Solicitudes: Los préstamos no son automáticos. Un administrador debe aprobar tu solicitud antes de que puedas retirar el recurso de la biblioteca.

Gestionar Mis Préstamos

  1. Ve a "Mis Préstamos"
  2. Aquí verás todas tus solicitudes y préstamos:
    • Solicitudes Pendientes: Esperando aprobación
    • Préstamos Activos: Recursos que tienes prestados
    • Historial: Préstamos anteriores
  3. Para préstamos activos puedes:
    • Ver la fecha de devolución
    • Solicitar renovación (si está permitido)
    • Ver detalles del recurso

Estados de Préstamos

Estado Descripción Qué hacer
Solicitado Solicitud enviada, esperando aprobación Esperar respuesta del administrador
Aprobado Solicitud aprobada, puedes retirar el recurso Ir a la biblioteca a retirar
Activo Recurso prestado, en tu poder Usar el recurso, devolver antes de la fecha límite
Vencido Pasó la fecha de devolución Devolver inmediatamente
Devuelto Recurso devuelto exitosamente Préstamo completado
Rechazado Solicitud no aprobada Ver motivo de rechazo

6.3 Participación en el Foro

Explorar el Foro Académico

  1. Haz clic en "💬 Foro" en el menú
  2. Explora las categorías disponibles organizadas por temas académicos
  3. Haz clic en una categoría para ver las publicaciones

Crear una Publicación

  1. Haz clic en "Crear Publicación"
  2. Completa el formulario:
    • Título: Título descriptivo de tu publicación
    • Categoría: Selecciona la categoría más apropiada
    • Contenido: Escribe tu publicación con detalles
  3. Haz clic en "Publicar"
⚠️ Moderación: Todas las publicaciones deben ser aprobadas por un administrador antes de ser visibles para otros usuarios. Mantén el contenido académico y respetuoso.

Participar en Discusiones

  1. Haz clic en una publicación para ver su contenido completo
  2. Lee los comentarios existentes
  3. Para agregar tu comentario:
    • Escribe tu comentario en el campo correspondiente
    • Haz clic en "Comentar"
    • Tu comentario también requiere aprobación

Ver Mi Actividad en el Foro

  1. Haz clic en "Ver mi actividad en el foro"
  2. Verás un resumen de tu participación:
  3. Pestañas disponibles:
    • Mis Publicaciones: Todas tus publicaciones (aprobadas y pendientes)
    • Mis Comentarios: Todos tus comentarios con enlaces al contexto
  4. Desde aquí puedes:
    • Ver tus publicaciones pendientes de aprobación
    • Editar tus publicaciones
    • Ver el contexto de tus comentarios

7. ❓ Preguntas Frecuentes

Preguntas Generales

¿Cómo recupero mi contraseña?

Contacta al administrador de tu institución. Por seguridad, solo los administradores pueden resetear contraseñas.

¿Puedo cambiar mi información personal?

Los datos personales solo pueden ser modificados por administradores. Si necesitas actualizar información, contacta a la administración.

Para Estudiantes

¿Por qué no veo una materia en "Mis Materias"?

Posibles razones:

  • No estás inscrito en un grupo activo
  • La materia no está asignada a tu plan académico
  • Hay un problema con tu inscripción - contacta al administrador

¿Por qué mi solicitud de préstamo fue rechazada?

Las solicitudes pueden ser rechazadas por:

  • Recurso no disponible
  • Tienes préstamos vencidos
  • Alcanzaste el límite de préstamos simultáneos
  • El recurso está reservado

¿Por qué no aparece mi publicación en el foro?

Todas las publicaciones requieren aprobación de un moderador. Esto puede tomar tiempo dependiendo de la disponibilidad del personal.

Para Profesores

¿Por qué no puedo finalizar un período?

Para finalizar un período se requiere:

  • Que todas las materias tengan actividades que sumen exactamente 100% en porcentajes
  • Que todos los estudiantes estén calificados en todas las actividades

¿Puedo cambiar una calificación después de finalizar el período?

No, una vez finalizado el período las calificaciones quedan bloqueadas. Si necesitas hacer cambios, contacta al administrador.

¿Cómo calculo los porcentajes de mis actividades?

Los porcentajes deben sumar exactamente 100% por período. Por ejemplo:

  • 2 Exámenes: 40% cada uno = 80%
  • 1 Proyecto: 15%
  • 1 Participación: 5%
  • Total: 100%

Para Administradores

¿Cómo asigno estudiantes a grupos?

Al crear o editar un estudiante, simplemente asigna el "Nivel de Grado" y "Grupo". El sistema creará automáticamente la inscripción activa.

¿Puedo eliminar un usuario?

Sí, pero ten cuidado con eliminar usuarios que tienen actividad académica registrada (calificaciones, préstamos, etc.) ya que esto puede afectar los reportes.

8. 🛠️ Soporte Técnico

Canales de Soporte

Si experimentas problemas técnicos con el sistema, sigue estos pasos:

Proceso de Soporte

  1. Verifica este manual: Muchas dudas están resueltas aquí
  2. Intenta cerrar y abrir sesión: Esto resuelve problemas menores
  3. Verifica tu conexión a internet: Asegúrate de tener conexión estable
  4. Contacta al administrador: Si el problema persiste

Información para Reportar Problemas

Cuando reportes un problema, incluye:

  • Tu rol en el sistema: Estudiante, Profesor, Administrador
  • Qué estabas intentando hacer: Acción específica
  • Qué pasó exactamente: Error o comportamiento inesperado
  • Navegador que usas: Chrome, Firefox, Safari, etc.
  • Captura de pantalla: Si es posible
  • Hora aproximada: Cuándo ocurrió el problema

Navegadores Recomendados

El sistema funciona mejor en:

Navegador Versión Mínima Compatibilidad
Google Chrome 90+ ✅ Excelente
Mozilla Firefox 88+ ✅ Excelente
Microsoft Edge 90+ ✅ Excelente
Safari 14+ ✅ Buena

Consejos de Seguridad

🔒 Mantén tu Cuenta Segura

  • No compartir credenciales: Tu usuario y contraseña son personales
  • Cerrar sesión: Especialmente en computadores compartidos
  • Reportar actividad sospechosa: Si notas algo extraño en tu cuenta
  • Usar conexiones seguras: Evita redes WiFi públicas para acceder al sistema

Actualizaciones del Sistema

El sistema se actualiza periódicamente para mejorar funcionalidades y corregir errores. Las actualizaciones importantes serán comunicadas con anticipación.

✅ Mejores Prácticas

  • Actualiza tu navegador: Mantén tu navegador actualizado
  • Limpia caché periódicamente: Esto previene problemas de visualización
  • Usa el sistema regularmente: Para mantenerte al día con las novedades
  • Lee las notificaciones: El sistema puede mostrar avisos importantes
